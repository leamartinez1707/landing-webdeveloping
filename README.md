# Landing Page - Leandro Martínez | Soluciones Digitales

Sitio web profesional para Leandro Martínez, desarrollador de software especializado en sistemas para talleres, automatización de procesos y desarrollo a medida para emprendimientos y pequeños negocios en Uruguay y Latinoamérica.

## 📋 Descripción del Proyecto

Este es una landing page moderna construida con **React + Vite**, optimizada para SEO y dirigida al mercado de Uruguay y Latinoamérica. 

### Características Principales:
- 🎨 Diseño moderno y responsivo con Tailwind CSS
- ⚡ Animaciones suaves con Motion (Framer Motion)
- 📱 Completamente mobile-first
- 🔍 SEO optimizado para búsquedas locales
- 📈 Estructura semántica HTML5
- 🌐 Multiregión (Uruguay, Argentina, Chile, México, Colombia)

## 🚀 Tecnologias Utilizadas

- **React 19** - Framework de UI
- **Vite** - Build tool rápido
- **Tailwind CSS 4** - Styling
- **Motion (Framer Motion)** - Animaciones
- **EmailJS** - Envío de emails
- **Lucide Icons** - Iconografía

## 🛠️ Configuración y Desarrollo

### Requisitos Previos
- Node.js 16+
- npm o yarn

### Instalación

```bash
# Clonar repositorio
git clone https://github.com/tu-usuario/landing-webdeveloping.git
cd landing-webdeveloping

# Instalar dependencias
npm install

# Iniciar servidor de desarrollo
npm run dev

# Build para producción
npm run build

# Preview de build
npm run preview
```
